--> Description

The Courses Website is a static web project developed using HTML and CSS to present educational information in an organized and user-friendly format. The website includes multiple pages to display courses, instructors, certifications, and contact details. It demonstrates basic web development concepts such as page structuring, navigation, and styling.

--> Features

Home page with website overview

Courses listing page

Teachers information page

Certificate details page

Contact and Join pages

Simple and clear navigation

--> Technical Details

Technologies: HTML, CSS

Design Type: Static website

Deployment: GitHub Pages

Tools: Visual Studio Code, GitHub Desktop

--> Conclusion

This project serves as an introductory example of static website development and showcases fundamental web design and content organization skills.
